# Твой Выбор — сайт доставки правильного питания

Статический сайт (HTML + CSS + JS). Никаких фреймворков, npm или сервера не нужно.

---

## Как открыть сайт

1. Скопировать **всю папку** на новый компьютер (важно — не отдельные файлы, а папку целиком)
2. Открыть файл `index.html` в браузере двойным кликом

> Для корректного отображения шрифтов нужен интернет (Google Fonts).
> Всё остальное работает полностью офлайн.

## Как добавить новые блюда в меню

1. Положить фото (`.jpeg` или `.jpg`, рекомендуется 600×300 px) в папку `images/menu/`
2. В `index.html` найти секцию `<!-- MENU -->` и скопировать один блок `.menu-card`
3. Заменить:
   - `src="images/menu/НАЗВАНИЕ.jpeg"` — путь к новому фото
   - `alt="..."` — название блюда
   - Тег программы (Салаты / Основное / Завтрак / Супы / Снеки)
   - Название, описание, КБЖУ и калории

## Как изменить FAQ

В `index.html` найти секцию `<!-- FAQ -->`. Каждый вопрос — это блок:

```html
<div class="faq-item reveal-up delay-1">
    <button class="faq-question" aria-expanded="false">
        <span>ТЕКСТ ВОПРОСА</span>
        <span class="faq-icon" aria-hidden="true"></span>
    </button>
    <div class="faq-answer">
        <div class="faq-answer-inner">
            <p>ТЕКСТ ОТВЕТА</p>
        </div>
    </div>
</div>
```

## Как изменить цвета

Все цвета — в начале `style.css` в блоке `:root`:

```css
:root {
    --bg-dark:      #0A1209;  /* основной фон */
    --gold:         #C8A050;  /* золотой акцент */
    --green:        #5A9E6F;  /* зелёный акцент (кнопки) */
    --text-primary: #F0EBE0;  /* основной текст */
}
```

## Шрифты

Сайт использует Google Fonts:
- **Cormorant Garamond** — заголовки
- **Jost** — основной текст

Подключаются автоматически при наличии интернета. Если нужна офлайн-версия — скачайте шрифты с fonts.google.com и подключите через `@font-face` в `style.css`.

## Если что-то не работает

- **Видео не воспроизводится** — проверьте, что файлы `video-hero.webm` и `video-hero.mp4` лежат в корне папки
- **Картинки в меню не отображаются** — файлы должны быть в папке `images/menu/` с расширением `.jpeg` (не `.jpg`)
- **Логотип не отображается** — файл `logo.png` должен быть в папке `images/`
- **Шрифты выглядят иначе** — проверьте подключение к интернету
